# Teslo API

1. Clonar proyecto
2. ```yarn install```
3. Clonar el archivo ```.env.template``` y renombrarlo a ```.env```
4. Cambiar las varibales de entorno
5. Levantar base de datos

```
docker-compose up -d
```

6. Ejecutar Seed
```
http://localhost:3001/api/seed
```

7. Levantar: ```yarn start:dev```
